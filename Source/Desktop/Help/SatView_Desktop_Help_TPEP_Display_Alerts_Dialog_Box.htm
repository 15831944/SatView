TPEP Parameter Alerts Dialog Box

SUMMARY

Saturday, October 10, 2015
Version 1.0

Telemetry Parameter Exchange Protocol Displays (TPEP) provide the capability to control the values of selected telemetry parameters and perform an alert on request.

How do I...

Alert Parameter Values

  1. Open the TPEP Alerts Dialog Box if it is not open yet by activating the TPEP display that should generate the alerts and by selecting the Alerts... item on the application button's menu. Pushing the Alerts button on the TPEP display's toolbar is another way to do this.
  2. Enter a name for the alert to be created into the Name edit field or select an existing one in the List list box to be modified.
  3. Select the telemetry parameter to be alerted in the Watch combo box and select the appropriate operator to the right of it.
  4. Enter the value to be compared against into the edit field right to the operators combo box.
    Note:
    For calibrated status parameters select one of the labels proposed in the combo box.
  5. Select the audition to be performed in the Audition combo box. Repeat this step for any notification to be done with the Notification combo box.
  6. Enter any text to be used for the audition or notification into the Text edit field.
  7. Push the Add button.
    Use the Remove button to delete an existing alert or Remove All to remove all of them.
  8. Push the OK button.

Note:
This option is available for normal TPEP displays only.

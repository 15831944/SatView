Telemetry Reports File Triggers Dialog Box

SUMMARY

Saturday, October 10, 2015
Version 1.0

The execution of telemetry report profiles can be triggered by system alerts, telemetry events, file manipulations and by a time schedule. This dialog box allows the specification of the file triggers associated with a telemetry report profile.

How do I...

Trigger Telemetry Report Profiles by Files

  1. Open the Telemetry Reports File Triggers Dialog Box if it is not open yet:
    Push the Reports button inside the Reports panel of the Telemetry ribbon bar category, select the telemetry report profile that should be triggered by files in the Name combo box of the dialog box and push the File Triggers... button.
  2. Enter the name of a directory to be monitored for any changes into the Directories edit field.
    Use the Browse... button to search for a specific directory.
  3. Choose the events related to the directory that should trigger the associated telemetry report profile by marking one of more of the following options:

    Option

    Description

    New Files

    The telemetry report profile is triggered whenever a new file is created in the directory specified previously.

    Updated Files

    A trigger is raised whenever an existing file is updated in the directory.

    Deleted Files

    An update of the telemetry report occurs whenever a file is deleted inside the directory.
  4. Specify the scope of the event to be monitored in the directory:

    Scope

    Description

    All Files

    The event to be monitored applies to all files in the specified directory.

    Selected Files

    The event applies to some selected files only:

    1. Enter the file name (wildcards are allowed) into the edit field of the list box.
      Use the Browse... to search for a file.
      Note:
      Enter the file name only i.e. without path.
    2. Push the < button to add the specified file to the list.
      Use the > button to remove a previously chosen file.
  5. Push the Add button.
    Repeat the previous steps (except for the first one) in order to monitor more than one directory or push the Remove button to delete the current directory from the watch list.
  6. Push the OK button.

Prerequisites:
The privilege 'Changing the telemetry reports' must be enabled to create, modify or delete a telemetry report profile.

TPEP Settings Dialog Box

SUMMARY

Saturday, October 10, 2015
Version 1.0

Many aspects of a Telemetry Parameter Exchange Protocol Display (TPEP) are configurable. Layout, filtering options and print jobs can be modified as required. Furthermore, it is also possible to define the way how telemetry data should be displayed with respect to validity and out-of-limit status colors. Tool-tips, providing details about telemetry parameters and, in particular, their values can be enabled or disabled as well.

How do I...

Modify the Layout

  1. Open the TPEP Settings Dialog Box if it is not open yet:
    Activate the TPEP display and select the Settings... item on the application button's menu. Pushing the Settings button on the TPEP display's toolbar is another way to do this.
  2. Select the Layout page.
  3. Specifying the Appearance:

    Creating a new Column

    1. Inside the Format group box choose the column to be inserted by selecting the corresponding name in the Name combo box.
    2. Click at the position on the column items bar below the Name combo box where the selected column should be inserted. A '*' indicating the selected column appears.
    3. Push the Insert Before or Insert After button.
      Note:
      The ID column must always be the left most one of a group i.e. all columns (until the next ID column) following this one to the right refer to it.

    Deleting a Column

    1. Select the column to be deleted by clicking on it in the column items bar below the Name combo box. A '*' indicating the selected column appears.
    2. Push the Remove button.
      Note:
      At least one ID column must be defined.

    Changing the Fonts

    1. Inside the Fonts group box select the Title or Messages radio button depending on which font should be changed. The Title radio button refers to title bar on top of each display and the Messages button to the messages listed.
    2. Select the font name in the Name combo box and select the desired style and size in the Style combo box and Size edit field respectively. A sample of the chosen font is always displayed in the Sample area.
    Controlling the Operating Flags:

    Tool-tips

    A small tool-tip containing telemetry parameter related details can be displayed on request. When this option is enabled and the cursor is moved over a parameter inside the TPEP display, a small hand occurs indicating that a tool-tip is available. Press the right mouse button to make it pop-up. The tool-tip disappears automatically when the mouse button is released.

    • Mark the Tooltips check box in order to enable this feature.
  4. Push the OK button.

Change the Filtering Options

  1. Open the TPEP Settings Dialog Box if it is not open yet:
    Activate the TPEP display and select the Settings... item on the application button's menu. Pushing the Settings button on the TPEP display's toolbar is another way to do this.
  2. Select the Filtering page.
  3. Filter by:

    Time

    • Mark the Between t1 and t2 check box and fill-in the time range t1 < t2 into the appropriate edit fields.

    Note:
    Messages or parameter values will be listed between t1 and t2 only.

    Daily Time

    • Mark the Daily between t1 and t2 check box and fill-in the daily time range into the appropriate edit fields.

    Note:
    Messages or parameter values will be listed daily between t1 and t2 only.

    Periodical Time Interval

    • Mark the Periodically every n minutes check box and fill-in the required period of n minutes into the appropriate edit field.
      Optional:
      • Mark the Starting at time t check box and fill-in the time t into the edit field if the periodical filtering should start at a specific time.
      • Mark the Stopping at time t check box and fill-in the time t into the edit field if the periodical filtering should stop at a specific time.

    Note:
    The listing of messages will take place in an intermittent way within the specified time intervals.

    Parameter Validity & Status

    • Mark the Status check box and select the status or validity that should force the telemetry parameter value to be listed.

    Note:
    This option applies to normal TPEP displays only and, if is not enabled, any parameter value will be listed.

    Parameter Quality

    • Mark the Quality check box and determine if only good or bad data should be listed by selecting the appropriate check boxes.

    Note:
    This option applies to normal TPEP displays only and, if is not enabled, any parameter value will be listed.
  4. Limit by:

    Message Count

    • Mark the A maximum of n messages check box if a maximum of n messages should be listed and enter the limit n into the corresponding edit field.

    Note:
    The number of values will be adjusted the next time a value is listed.

  5. Specifying the Number of Digits

    • Mark the Digits check box if numbers should be limited to an amount of n digits and enter the limit n into the appropriate edit field.

    Note:
    This option applies to floating-point numbers inside normal TPEP displays only.

    Specifying the Color of the Messages

    • Select the desired color x for the contents to be listed in the For simple messages or parameter information use x color combo box.

    Modifying the Parameter Status Colors

    • Select the status x for which the color should be changed in the For values with status x use color y combo box and choose the color y in the other one.

    Note:
    This option applies to normal TPEP displays only.
  6. Push the OK button.

Use Print Job Profiles

  1. Open the TPEP Settings Dialog Box if it is not open yet:
    Activate the TPEP display and select the Settings... item on the application button's menu. Pushing the Settings button on the TPEP display's toolbar is another way to do this.
  2. Select the Printing page.

  3. Creating a new Print Job Profile

    1. Type the name of the print job profile to be created into the Name combo box.
      Note:
      The Default print job profile is used when printing with the quick printing method and can only be modified but not removed.
    2. In the Properties group box specify the print profile to be used with this print job profile by selecting the corresponding profile in the Print Profile combo box. The print profile specifies the printer and paper related properties whereas the print job profile adds some, more TPEP display specific, printing directives.
    3. Specify if all values or only the first/last n values should be printed by checking either the Print all messages or Print the first/last n messages radio button and by filling in the number n in the edit field.
    4. Mark the Underline OOLs check box if an out-of-limit value should be underlined.
      Note:
      This option is useful when printing without colors.
    5. Mark the Print OOL Legend check box if a small legend should be included on the print-out indicating the out-of-limit status color association.
      Note:
      This option is useful when printing in colors.
    6. Type a comment into the Text edit field if additional information should be added to the title of the print-out. Mark the Clear after use check box if this text should be used in a single print-out only.
    7. Push the Add button.

    Modifying a Print Job Profile

    • Select the name of the print job profile to be modified in the Name combo box and perform the necessary changes (see 'Creating a new Print Job Profile').

    Deleting a Print Job Profile

    1. Select the name of the print job profile to be deleted in the Name combo box.
    2. Push the Remove button to remove the selected print job profile. Use the Remove All button to delete all existing profiles.
      Note:
      The Default print job profile cannot be deleted.
  4. Push the OK button.

Schedule Print Jobs

  1. Open the TPEP Settings Dialog Box if it is not open yet:
    Activate the TPEP display and select the Settings... item on the application button's menu. Pushing the Settings button on the TPEP display's toolbar is another way to do this.
  2. Select the Jobs page.
  3. Select the print job to be scheduled in the Defined list box.
  4. Specify when the print job should be executed by marking one or more of the available options:

    Option

    Description

    Print at YYYY.DDD.hh:mm:ss

    Prints the TPEP display at a specified time.
    Type the time into the associated time edit field.

    Print at hh:mm:ss every day

    Prints the TPEP display every day at a specified time.
    Type the time into the associated time edit field.

    Print at name event

    Prints the TPEP display when the event name is in a signaled state.
    Select the event in the associated combo box.
  5. When printing at a certain time is enabled it is possible to specify how many times the printing should be repeated and with what interval. In the For repetition type combo box select the repetition characteristics for either daily or non-daily printing. In the case of non-daily printing specify the interval in minutes after which another printing should occur by checking the Repeat every n minutes and by typing the interval n into the associated edit field.
    For both kinds of printing stop conditions can be specified by selecting either the Stop after n copies or Stop at time YYYY.DDD.mm:hh:ss radio button and by filling in the appropriate data into the edit fields.
  6. Push the > button to activate the print job. The name of the print job is copied from the Defined list box into the Active one in order to indicate the activation.
    Whenever a print job needs to be deactivated but not deleted select it in the Active list box and then push the < button.
    Note:
    Deactivated print jobs are not scheduled.
  7. Push the OK button.

Telemetry Events Dialog Box

SUMMARY

Saturday, October 10, 2015
Version 1.0

SatView™ supports a mechanism to allow users to be notified in various ways when a certain predefined telemetry event occurs. Such an event can be identical to a telemetry parameter smaller, equal, non-equal or bigger than a constant value or a more complex expression consisting of any thinkable mathematical calculation (comparable to a derived parameter). Any existing audition and notification profile can be attached to such an event.

How do I...

Create an Event

  1. Open the Telemetry Events Dialog Box if it is not open yet:
    Push the Events button inside the Events panel of the Telemetry ribbon bar category.
  2. Type the name of the new telemetry event into the Name edit field.
  3. Construct the expression to be evaluated by considering the following points:
    • Construct the expression in Reverse Polish Notation.
    • The expression must be normalized i.e. comparable against zero.
    • A small red horizontal bar indicates the current insertion point on the stack.
    • Use the é or ê button to move that point up and down the stack.
      Use the ç or è button to add or remove an element (i.e. parameter, operator, function or number) on the stack.
    • Past parameter values can also be specified by entering a negative sample value into the Sample edit field:
      A sample number of -1 for example specifies the value preceding the current one and -2 even one before. The expression, however, is evaluated only when all values (including the past ones) are available.

  4. Choose the final comparison to be performed with the result of the expression evaluation in the Result combo box.
  5. Enable one or more of the following options if required:

    Option

    Description

    Invalid Data

    The telemetry event is evaluated even when one or more of the contained parameters are invalid (i.e. their validity parameter is false).

    Bad Data

    The telemetry event is evaluated even when one or more of the contained parameters are marked as potentially illegal (received inside a telemetry packet or format marked as bad).

    Disabled

    Suspends the evaluation of the telemetry event.
  6. Choose the audition to be performed when the expression evaluation returns true in the Audition combo box.
  7. Choose the notification to be executed when the telemetry event becomes triggered in the Notification combo box.
  8. Enter the text to be used for the notification into the Text edit field.
    Note:
    Attaching a text is optional if neither an audition nor a notification is selected.
  9. Push the Add button.
    Repeat the previous steps in order to add more than one telemetry event.
  10. Push the OK button.

Prerequisites:
The privilege 'Changing the telemetry events' must be enabled to create, modify or delete a telemetry event.

Modify an Event

  1. Open the Telemetry Events Dialog Box if it is not open yet:
    Push the Events button inside the Events panel of the Telemetry ribbon bar category.
  2. Select the name of the telemetry event to be modified in the Name combo box.
  3. Position the insertion point to the area on the stack that needs to be changed by using the é or ê keys and perform the necessary modifications.
  4. Push the OK button.

Prerequisites:
The privilege 'Changing the telemetry events' must be enabled to create, modify or delete a telemetry event.

Delete an Event

  1. Open the Telemetry Events Dialog Box if it is not open yet:
    Push the Events button inside the Events panel of the Telemetry ribbon bar category.
  2. Select the name of the telemetry event to be deleted in the Name combo box.
  3. Push the Remove button. Use the Remove All button in order to delete all telemetry events at once.
  4. Push the OK button.

Prerequisites:
The privilege 'Changing the telemetry events' must be enabled to create, modify or delete a telemetry event.

Import Events

  1. Open the Telemetry Events Import Dialog Box if it is not open yet:
    Push the Import... button inside the Events panel of the Telemetry ribbon bar category.
  2. Enter the name of the file that contains all telemetry events to be imported into the Filename edit field.
    Use the Browse... button in order to search for an appropriate source file if necessary.
  3. Push the Inspect button in order to list all events that are contained in the specified file.
    Note:
    The Inspect button is available only when a file name is specified. With an empty Filename edit field the Browse... button is visible instead.
  4. In the Directory combo box select a telemetry event that should be imported.
  5. Push the Add button. Repeat the previous two steps if more than one telemetry event should be imported or use the Add All button in order to import all available ones.
    Use the Remove or Remove All button in order to remove one or several telemetry events from the importation list.
    Note:
    Telemetry events can be imported from more than one source file in one step by entering a different file name into the Filename edit field and following the steps above.
  6. Push the OK button.

Prerequisites:
The privilege 'Importing telemetry events' must be enabled to import telemetry events.

Export Events

  1. Open the Telemetry Events Export Dialog Box if it is not open yet:
    Push the Export... button inside the Events panel of the Telemetry ribbon bar category.
  2. In the Directory combo box select a telemetry event that should be exported.
  3. Push the Add button. Repeat the previous steps if more than one telemetry event should be exported or use the Add All button in order to export all available ones.
    Use the Remove or Remove All button in order to remove one or several telemetry events from the exportation list.
  4. Enter the name of the file that should contain all exported telemetry events into the Filename edit field.
    Use the Browse... button in order to search for an appropriate destination file if necessary.
  5. Push the OK button.

Prerequisites:
The privilege 'Exporting telemetry events' must be enabled to export telemetry events.

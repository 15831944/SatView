Telemetry Data Archival Options Dialog Box

SUMMARY

Saturday, October 10, 2015
Version 1.0

SatView™ performs archiving of telemetry data on the basis of normal files. Multiple files can be specified and each of those files can be configured separately with respect to its properties and behavior. Furthermore, some effort has been made concerning automatic backups and to assure that no telemetry data is lost in case of failures.

How do I...

Specify Archival Retention Options

These archival retention options are available to find an optimal trade-off between space requirements and retrieval efficiency and they apply to all files in the archives. They control the contents of the full updates which are always performed after n telemetry source packets or formats. The archiving of the raw telemetry source packet stream however is not affected.
It should also be pointed out that by specifying a limited number of n days to archive a telemetry source packet or telemetry parameter that data will be lost for any retrievals starting n days after their last update.

  1. Open the Telemetry Data Archival Options Dialog Box if it is not open yet:
    Push the Settings button inside the Archives panel of the Telemetry ribbon bar category, select the Archivals page in the dialog box and push the Options... button.
  2. Specify the archiving behavior for all or a specific telemetry source packet (or format) by selecting either the item All telemetry source packets or the name of an individual one in the upper combo box.
  3. Push either the Forever radio button to keep the chosen telemetry source packets (or formats) in the cache all the time or for a limited amount of days only by checking the n days only radio button and entering the number n of days.
    Note:
    Specifying a retention of 0 days does not archive the telemetry source packet at all (in the full cache updates).
  4. Repeat the previous two steps in order to specify the retention options for more than one specific telemetry source packet (or format).
    Note:
    The current settings can always be reset by re-selecting the item 'All telemetry source packets' and by specifying the appropriate option.
  5. Specify the archiving behavior for all or a specific telemetry parameter by selecting either the item All telemetry parameters or the name of an individual one in the lower combo box.
  6. Push either the Forever radio button to keep the chosen telemetry parameters in the cache all the time or for a limited amount of days only by checking the n days only radio button and entering the number n of days.
    Note:
    Specifying a retention of 0 days does not archive the telemetry parameter at all (in the full cache updates).
  7. Repeat the previous two steps in order to specify the retention options for more than one specific telemetry parameter.
    Note:
    The current settings can always be reset by re-selecting the item 'All telemetry parameters' and by specifying the appropriate option.
  8. Push the OK button.

Prerequisites:
The privilege 'Changing the telemetry data archive settings' must be enabled to configure the telemetry data archives.

TC Procedures Monitor Release Options Dialog Box

SUMMARY

Saturday, October 10, 2015
Version 1.0

The TC Procedures Monitor controls the execution of telecommand procedures. This dialog box is designed to specify release constraints for any telecommand of a certain telecommand procedure. It also allows the association of interlocks and release time windows to a telecommand.

How do I...

Specify the Release Constraints for Telecommand Procedures

  1. Open the TC Procedures Monitor Settings Dialog Box if it is not open yet:
    Activate the TC Procedures Monitor and select the Settings... item on the application button's menu. Alternatively, push the Settings button on the display's toolbar.
  2. Select the Contents page.
  3. Choose the telecommand procedure those release constraints should be changed in the contents list box.
  4. Make sure that the Command Source check box is not selected and push the Commands... button.
    The TC Procedures Monitor Release Options Dialog Box will occur.
  5. Mark the Interlock check box in order to force the use of interlocks by telecommands of the procedure and choose that interlock in the associated combo box.
    Note:
    New interlocks can be created or existing ones can be modified or deleted by pushing the Interlocks... button.
  6. Specify the desired release time window to be used by the telecommand procedure by checking either the Default radio button (uses the interval specified in the Telecommand Options Dialog Box), Unlimited or Limited to n seconds radio button. Enter the number of seconds n into the related edit field if a limited time window is chosen.
  7. Push all necessary OK buttons.

Note:
Any specified release constraints will be valid for all instances of a telecommand procedure.

Prerequisites:
The privileges 'Changing the contents of the telecommand procedures monitor' or 'Performing privileged tasks on the telecommand procedures monitor' must be enabled to specify or modify release constraints.

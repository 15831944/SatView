System Options

SUMMARY

Saturday, October 10, 2015
Version 1.0

SatView™ provides an interface to configure database update options, to setup, enable/disable the integrated satellite tracking or to specify the e-mail settings.

How do I...

Configure Database Options

  1. Open the System Options Dialog Box if it is not open yet:
    Push the Options button inside the Customization panel of the Tools ribbon bar category.
  2. Select the Database page.
  3. Select the desired option:

    Option

    Description

    Do nothing

    No action is performed when a new release of the current database is available.

    Inform with an event

    An event is generated and displayed in the Global Eventbox in order to inform about the availability of a new database release.

    Update the database immediately

    The current database is updated immediately after a new release has become available.
    Note:
    The telemetry data processing will be interrupted while the database update takes place.

    Update the database between t1 and t2 only

    Any automatic database will be delayed to take place at the time interval specified.
    A possible value t2 < t1 means that updates will only take place from t1 to midnight to t2 (next day) whereas a range t1 ≤ t2 allows updates between t1 and t2 only.
    Note:
    The telemetry data processing will be interrupted while the database update takes place.

    Note:
    Mark the Update the database now check box in order to force a database update manually (this option is only enabled if a new release is available).

  4. Push the OK button.

Prerequisites:
The privilege 'Changing the database update options' must be enabled to configure the database update options.

Configure Satellite Tracking Options

  1. Open the System Options Dialog Box if it is not open yet:
    Push the Options button inside the Customization panel of the Tools ribbon bar category.
  2. Select the Satellite Tracking page.
  3. Enter the path of the human-readable configuration file of the satellite tracking tool into the Filename edit field.
    Note:
    This file is usually located in the same directory as the executable files.
  4. Enter the path of the help file of the satellite tracking tool into the Helpfile edit field.
  5. Mark the Enabled check box if the satellite tracking tool should be enabled and running.
    Note:
    Disabling the satelite tracking tool can save a significant amount of CPU time, in particular on low-end systems.
  6. Push the OK button.

Prerequisites:
The privilege 'Changing the satellite tracking options' must be enabled to configure the satellite tracking options.

Configure E-Mail Options

  1. Open the System Options Dialog Box if it is not open yet:
    Push the Options button inside the Customization panel of the Tools ribbon bar category.
  2. Select the E-Mail page.
  3. Enter the address of the mail server into the Server edit field.
  4. Choose between the various available connection types by checking either the Plain text, SSL or TLS, STARTTLS or Auto STARTTLS radio button.
  5. Specify the port to be used for the communication with the mail server by entering the corresponding number into the Port spin box.
  6. Additional e-mail configuration options can be set or modified by pushing the Advanced... button.
    Note:
    Consult third-party documentation to learn about the supported e-mail configuration options.
  7. Account details can be specified by entering the account name into the Name edit field and by filling-in the address, log-in user name & password into the Address, Log-in & Password edit fields inside the Account group box.
  8. Push the OK button.

Prerequisites:
The privilege 'Changing the e-mail options' must be enabled to configure the e-mail options.

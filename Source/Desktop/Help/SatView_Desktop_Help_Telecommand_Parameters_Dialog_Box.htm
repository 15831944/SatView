Telecommand Parameters Dialog Box

SUMMARY

Saturday, October 10, 2015
Version 1.0

SatView™ provides a convenient interface to view the properties of telecommand parameters. Type, nature, coding of a parameter as well as other relevant information is displayed. It is also possible to initialize them with a value which is particularly useful for preparing telecommands.

How do I...

Browse Parameters

  1. Open the Telecommand Parameters Dialog Box if it is not open yet:
    Push the View button inside the Parameters panel of the Telecommand ribbon bar category.
  2. Select the telecommand parameter to be browsed in the Parameter combo box.
  3. View all details inside the various pages:

    Page

    Contents

    Characteristics

    Shows details about the type, nature, coding and radix of the telecommand parameter.

    Calibration

    Shows all calibration curves associated with the telecommand parameter. As there may be more than one, it also indicates which is the current (active) calibration that is applied.
    Note:
    This page is blocked (i.e. cannot be selected) in cases where the selected telecommand parameter has no attached calibration curve.

    Limits

    Shows all out-of-limit tables associated with the telecommand parameter. As there may be more than one, it also indicates which is the current (active) table that is applied.
    Note:
    This page is blocked (i.e. cannot be selected) in cases where the selected telecommand parameter has no attached out-of-limit table.

    Allocations

    Identifies the telecommand source packets or formats that contain the selected telecommand parameter.
  4. Push the OK button.

Initialize Parameters

  1. Open the Telecommand Parameters Dialog Box if it is not open yet:
    Push the View button inside the Parameters panel of the Telecommand ribbon bar category.
  2. Select the telecommand parameter that needs to be initialized (or those value needs to be changed) in the Parameter combo box.
  3. Push the Add button inside the Initializations group box.
  4. Specify how the value that is going to be entered has to be interpreted in the Interpretation combo box:

    Interpretation

    Description

    Calibrated

    The value has to be interpreted as calibrated data. This will cause an automatic de-calibration to be performed.
    Note:
    This option is not available for telecommand parameters having no attached calibration curves.

    Coded

    The value has to be interpreted as uncalibrated (raw) data using the coding and radix specified for the telecommand parameter.
    Example: 2.0 = 2.0 (with coding: floating-point, radix: decimal).

    Raw (Hex)

    The value has to be interpreted as raw data in hexadecimal format.
    Example: 2.0 = 40000000 (with coding: floating-point, radix: hexadecimal).
  5. Enter the desired value into the Value edit field.
    Note:
    For telecommand parameters with a textual calibration curve attached all possible values are listed in the associated combo box.
  6. Push the Initialize button.
    Repeat the previous steps if more than one telecommand parameter should be initialized.
  7. Push the OK button.

Prerequisites:
The privilege 'Changing the value of telecommand parameters' must be enabled to initialize or change the value of a telecommand parameter.

Reset all Parameters

  1. Open the Telecommand Parameters Dialog Box if it is not open yet:
    Push the View button inside the Parameters panel of the Telecommand ribbon bar category.
  2. Push the Reset All button.
  3. Push the OK button.

Prerequisites:
The privilege 'Changing the value of telecommand parameters' must be enabled to reset telecommand parameters.

Create a new Parameter Set

Telecommand parameter sets are used to combine initializations of several parameters at once.

  1. Open the Telecommand Parameters Dialog Box if it is not open yet:
    Push the View button inside the Parameters panel of the Telecommand ribbon bar category.
  2. Add all telecommand parameters that should be part of the set to the Initializations group box and initialize all of them with the desired values (see above how to do that).
  3. Enter the name of the set into the Set edit field.
  4. Push the Save button.
  5. Push the OK button.

Prerequisites:
The privilege 'Changing the value of telecommand parameters' must be enabled to create, modify or delete a telecommand parameter set.

Modify a Parameter Set

Telecommand parameter sets are used to combine initializations of several parameters at once.

  1. Open the Telecommand Parameters Dialog Box if it is not open yet:
    Push the View button inside the Parameters panel of the Telecommand ribbon bar category.
  2. Select the set to be modified in the Set combo box.
  3. Add additional telecommand parameters or remove existing ones from the Initializations list box by pushing the Add or Remove button.
  4. Push the Save button.
  5. Push the OK button.

Prerequisites:
The privilege 'Changing the value of telecommand parameters' must be enabled to create, modify or delete a telecommand parameter set.

Delete a Parameter Set

Telecommand parameter sets are used to combine initializations of several parameters at once.

  1. Open the Telecommand Parameters Dialog Box if it is not open yet:
    Push the View button inside the Parameters panel of the Telecommand ribbon bar category.
  2. Select the set to be deleted in the Set combo box.
  3. Push the Delete button.
  4. Push the OK button.

Prerequisites:
The privilege 'Changing the value of telecommand parameters' must be enabled to create, modify or delete a telecommand parameter set.

Import Parameter Sets

  1. Open the Telecommand Parameters Dialog Box if it is not open yet:
    Push the View button inside the Parameters panel of the Telecommand ribbon bar category.
  2. Push the Import... button. This opens the Telecommand Parameter Sets Import Dialog Box.
  3. Enter the name of the file containing the set into the Filename edit field.
    Use the Browse... button to search for the file if necessary.
  4. Push the Inspect button to check the contents of the file which is shown in the Import list box.
  5. Remove those sets that should not be imported by selecting them first in the Import list box and then pushing the Remove button. Use the Remove All button to clear the list box.
  6. Push the OK button.

Prerequisites:
The privilege 'Changing the value of telecommand parameters' must be enabled to import telecommand parameter sets.

Export Parameter Sets

  1. Open the Telecommand Parameters Dialog Box if it is not open yet:
    Push the View button inside the Parameters panel of the Telecommand ribbon bar category.
  2. Push the Export... button. This opens the Telecommand Parameter Sets Export Dialog Box.
  3. Select the set to be exported in the Sets combo box.
  4. Push the Add button.
    Repeat the previous two steps if more than one set should be exported or use the Remove button to clear a selected one from the exportation list.
  5. Enter the name of the file to receive the exported sets into the Filename edit field.
    Use the Browse... button to search for an existing file if necessary.
  6. Push the OK button.

Prerequisites:
The privilege 'Changing the value of telecommand parameters' must be enabled to export telecommand parameter sets.

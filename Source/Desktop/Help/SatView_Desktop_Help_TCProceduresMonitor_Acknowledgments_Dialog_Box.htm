TC Procedures Monitor Acknowledgments Dialog Box

SUMMARY

Saturday, October 10, 2015
Version 1.0

The TC Procedures Monitor controls the execution of telecommand procedures which can be in an alerted state. This dialog box shows all alerts related to a telecommand procedure and allows them to be acknowledged in a 'last-in-first-out' way.

How do I...

Acknowledge alerted Telecommand Procedures

Telecommand procedures that are in an alerted state blink inside the summary pane and indicate the problem with a small icon on the left side. An attached tool-tip can be activated on that symbol to view more details.

  1. Select the telecommand procedure to be acknowledged inside the summary pane.
  2. Select the Details... item on the application button's menu. Alternatively, push the Details button on the display's toolbar.
    Either the TC Procedures Monitor Dialog Box or TC Procedures Monitor Acknowledgments Dialog Box will occur.
  3. Push the Acknowledge button.
    Repeat the previous steps if more than one alert needs to be acknowledged.
    Note:
    Each telecommand procedure maintains a stack of alerts which might require multiple acknowledgments.
  4. Push the OK button.

Telemetry Parameter Sets Import Dialog Box

SUMMARY

Saturday, May 09, 2015
Version 1.0

SatView™ provides a convenient interface to view the properties of telemetry parameters. For constants and derived parameters it is also possible to modify the current value. This is particularly useful for iterative derived parameters which always need an initialization. Repetitive initializations of telemetry parameter groups can be simplified by the capability to use sets which can be imported and exported.

How do I...

Import Parameter Sets

  1. Open the Telemetry Parameter Sets Import Dialog Box if it is not open yet:
    Push the View button inside the Parameters panel of the Telemetry ribbon bar category and then push the Import... button in the dialog box.
  2. Enter the name of the file containing the set into the Filename edit field.
    Use the Browse... button to search for the file if necessary.
  3. Push the Inspect button to check the contents of the file which is shown in the Import list box.
  4. Remove those sets that should not be imported by selecting them first in the Import list box and then pushing the Remove button. Use the Remove All button to clear the list box.
  5. Push the OK button.

Prerequisites:
The privilege 'Changing the value of telemetry parameters' must be enabled to import telemetry parameter sets.

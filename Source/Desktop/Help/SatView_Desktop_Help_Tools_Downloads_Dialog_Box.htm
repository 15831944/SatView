Tools Downloads Dialog Box

SUMMARY

Saturday, October 10, 2015
Version 1.0

SatView™ supports scheduled or triggered downloads of files. This is a useful feature to acquire information from remote sources that are necessary for further processing at a certain time or event.

How do I...

Schedule or trigger Downloads

  1. Open the Tools Downloads Dialog Box if it is not open yet:
    Push the Settings button inside the Downloads panel of the Tools ribbon bar category.
  2. Enter or select the name of the download profile to be created or modified in the Profile combo box.
  3. On the Files page:
    • Enter the name of the files to be downloaded into the Files edit field.
      Note:
      • Multiple files must be separated by a semicolon.
      • Wildcards (e.g. input*.*) as part of the file name are allowed. In this case an automatic search for the files will be initiated.
    • Specify a destination directory for the downloaded files.
      Note:
      The temporary directory is used as target location for any files that need to be downloaded if no directory is specified.
    • Enter the details about the FTP location from where to download the requested files into the FTP Location, Username, Password edit fields and specify the Port used if necessary.
    • Specify the transfer mode by choosing either the ASCII Mode radio button or Binary Mode radio button.
      Note:
      The transfer mode applies to all files downloaded within the selected profile.
  4. On the Schedule page:
    • Choose one of the available execution modes by checking the appropriate radio button:

      Mode

      Description

      Immediately

      The download process is started immediately.

      Once at t

      The download is executed at time t.

      Daily at t

      The download is executed daily at time t.

      Starting at t1 with an interval of n minutes and stopping at t2

      The execution of the download takes place periodically with an interval of n minutes, starting at time t1 and stopping at t2.

      At event x with a delay of n minutes

      Executes the download profile n minutes after the event x occurred.
  5. Push the Save button to save the modifications of the profile.
    Use the Delete button in order to remove an existing profile or the Delete All button to remove all of them at once.
    Note:
    The Default download profile can never be deleted. It is reset whenever the Delete button is pushed.
  6. Push the OK button.

Prerequisites:
The privilege 'Changing the tools download profiles' must be enabled to create, modify or delete download profiles.
